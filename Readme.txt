Web_server Folder : เป็นโฟลเดอร์ที่ใช้เก็บส่วนของเว็บแอปพลิเคชันทั้งหมด โดยภายในประกอบด้วย 2 โฟลเดอร์ย่อย ได้แก่ 
	(1) backend : เป็นโฟลเดอร์ที่เกี่ยวกับระบบหลังบ้านทั้งหมด เช่น ส่วนที่ใช้ประมวลผลคำสั่ง การเชื่อมต่อฐานข้อมูล การจัดการ API การตรวจสอบสิทธิ์ผู้ใช้งาน และฟังก์ชันทั้งหมดที่ทำให้ระบบสามารถทำงานได้จริง 
	(2) frontend : เป็นโฟลเดอร์ที่เกี่ยวกับระบบหน้าบ้านทั้งหมดหรือส่วนที่ผู้ใช้งานมองเห็น เช่น หน้าเว็บต่างๆ ฟอร์มการทำข้อสอบ หน้า dashboard รวมถึงส่วนที่ใช้ติดต่อกับ backend ผ่าน API ทั้งหมด

Typhoon_server Folder : เป็นโฟลเดอร์ที่ใช้เก็บโมเดล Typhoon LLM ซึ่งเป็นโมเดลที่ระบบนำมาใช้สำหรับการประเมินคำตอบอัตนัย การให้คะแนน และการประมวลผลข้อความตามที่ระบบต้องการ 

Database Folder : เป็นโฟลเดอร์ที่ใช้เก็บไฟล์ฐานข้อมูลทั้งหมดของระบบ เช่น ข้อสอบ คำตอบ คะแนน รวมถึงรายชื่อนักศึกษาทั้งหมดที่ถูกจัดเก็บไว้ในระบบ
